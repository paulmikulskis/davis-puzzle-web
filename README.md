Minecraft Pixel Puzzle Web is a tiny stateless Next.js app that lets an OT facilitator type one Minecraft item name, fetch the matching minecraft.wiki inventory texture through a no-store proxy, extract a browser-side palette, and download the same four-page Letter PDF worksheet format as the Python reference. Deployed URL: https://davis-puzzle-web.vercel.app. Credits: Andrew Davis designed the original worksheet format; this is an unofficial fan-made tool, with textures © Mojang and sourced from minecraft.wiki.
